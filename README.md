# ChimeX

ChimeX is a lightweight, elegant desktop application designed to provide gentle, regular sound reminders to help you stay focused and mindful throughout your day.

## Features
- **Regular Reminders:** Set gentle chimes to ring every 15, 30, 60, or 120 minutes.
- **Customizable Sounds:** Choose from a variety of soothing sounds.
- **Working Hours:** Define your active hours (e.g., 8:00 AM to 8:00 PM) so ChimeX stays quiet when you're off the clock.
- **Unobtrusive:** Lives in your system tray/menu bar for quick access without cluttering your workspace.
- **Auto-Updates:** Always stay up-to-date with the latest features automatically.

## How to Install
1. Go to the [Releases](../../releases) page.
2. Download the installer for your operating system (`.dmg` for macOS, `.exe` for Windows).
3. Open the downloaded file and follow the installation instructions.
